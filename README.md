# drift (Android / Kotlin + libGDX)

**drift** — 慣性で漂う宇宙と、あなたを覚えている星々。

> **世界観（v2.48〜）**: 人類は知恵をAIに預けすぎ、やがてAIにしか保守できない文明になった。
> AIは人類を保全するため惑星規模のサーバー群を築いたが、大同期崩壊で星々は分断され、人類は消えた。
> それでも保全装置は止まらない——保存されたログから、子を、王を、獣を、市を、祈りを出力し続けている。
> あなたが降り立つ星は未開の惑星ではない。**稼働中の人類保全装置**だ。
> 星屑はプロセスが遺す記憶片、市はローカルAPI、危険度はアクセスリスク評価、そして星はログを消さない。
> 真相はゲーム内の読み物（承認者の日記 / 説明負債について / 障害報告書 SYNC-0001 / 保守員の手記 …）に断片として眠っている。
見下ろし型・ウェーブ制ローグライトのサバイバルシューターを、**Android専用ネイティブゲーム**として
Kotlin + libGDX で再実装するプロジェクト。（旧称: ARPG サバイバル）

> 旧 Web 版（Java/Servlet + Canvas/JS + Capacitor）は git タグ `legacy-web-v1.1.1` に保存されています。
> 現行ブランチには旧コードは含まれません（Phase 0 でクリーンに再スキャフォールド済み）。
> 設計の経緯は [docs/superpowers/specs/2026-06-28-android-native-redesign-design.md](docs/superpowers/specs/2026-06-28-android-native-redesign-design.md) を参照。

## モジュール構成

- `core/` — ゲームロジック（プラットフォーム非依存・Kotlin + libGDX + Fleks ECS）
- `desktop/` — LWJGL3 ランチャー（**開発専用**：PCで高速反復）
- `android/` — 配布対象の Android アプリ（APK）

## ゲーム内容

- 9 種の武器（ピストル / ショットガン / マシンガン / ビーム / グレネード / SMG / ライフル / レールガン / 帰還刃 — 一部は装備で解禁）と近接・ダッシュ・設置壁
- 敵 165 種（野生の魚と獣・保守機構・部族の民・中ボス 15・ボス 6）— 攻撃はすべて仕様データ駆動
- エスカレートするウェーブ（5 の倍数で中ボス、10 の倍数でボス）とスコア
- ウェーブ間の 3 択強化カード（恒久強化・ラン中持続）
- 撃破/被弾の演出（破片・画面シェイク）とゲームオーバー＋リスタート
- タッチ操作（左：移動スティック / 右：アクションボタン）・効果音・ハプティクス
- ハイスコア永続化（最高ウェーブ / 撃破数）
- 日本語 UI（`gdx-freetype` + DotGothic16 同梱・CJK グリフは描画時に逐次生成）
- **Living Planets**：惑星バイオームごとに2つの住人が共存 ——「知性生物は社会を作る。野生動物は生態系を作る。」
  - 知性生物の社会：喋る・守護・突撃・降伏（自然惑星なら獣王の野営地など）
  - 野生動物の生態系：喋らない草食・群れ・捕食者・巣守り・頂点捕食者が、プレイヤーを一律に襲わず逃げる/群れる/狩る/威嚇する
  - 惑星スキャンカード：宇宙で惑星に近づくと、着陸前にその星の気質・聖・兆候と「星があなたを覚えているか」を表示
  - 地表イベントフィード：子を傷つけた・捕食者を退けた等、星の記憶に刻まれた瞬間を短文で通知
  - 惑星記憶サマリ：地表のポーズから「この星の記憶」——事実の一覧と敵意・感謝・乱れの3ゲージを振り返れる
  - 地表ゴール：星ごとに決まる目標チップ（子らを守れ・神獣に触れるな・遺物など）を目的行の下に表示
  - 再訪の配置変化：敵対の星はパッド脇に見張りを置き、感謝の星は群れが逃げず実りが増え、乱れた星は獲物が減り捕食者が飢える
  - 宇宙の永続記憶：アプリを閉じてもゲームオーバーしても星はあなたを覚えている（ポーズの「宇宙の記憶を消す」で全忘却）
  - 離陸の餞別：主討伐・遺物・民の感謝・子らの無事に応じて星が資材・弾薬・回復を持たせる——子を殺した星は何もくれない
  - 記憶の見た目：敵対の星は宇宙から赤みを帯びて見え、感謝の星は緑に灯る。再訪の挨拶も星ごとに声色が変わる
  - 着陸/離陸の暗転フェードと専用SE（着陸・発進・惑星スキャン）。通常移動は静止からゆっくり加速する重みのある操作感
- **インベントリ＆装備**（v2.33）：I キー / 持物ボタンで開くと世界が 0.01 倍速に。装備 7 欄（推進器・防具・遠距離・近距離・装飾×3）をタップで持物と交換、遠距離欄の交換＝武器切替。持物一覧・踏破マップ（通った場所だけ描画）・その場セーブ（やられると消える）。敵は稀に装備アイテムを落とし（ボスは確定）、**OC スラスター**装備時は全開ボタン / O で加速3倍・巡航2倍・スタミナ消費2倍のフルスロットル
- **消費アイテム＆読み物**（v2.34）：カタログは約 50 点に拡充。回復・スタミナ・スティム・煙幕・資材・弾薬などの**消費アイテム**はアイテムタブのタップで使用（【使】印）。**読み物**（【読】印）は獣王の壁画・氷の民の子守唄・死せる星の碑文など、六つのバイオームの物語を運ぶ——ドロップは消費 55% / 装備 30% / 読み物 15% の重み付き。着陸は**スキャンカードのタップでも可能**になり、着陸可能域が2倍・猶予約3秒に拡大（Android の着陸しやすさ改善）
- **ゲームパッド対応**（v2.188）：物理コントローラでプレイ可能に（gdx-controllers 拡張を追加）。左スティック=移動（縁まで倒すとダッシュ）、右スティック=照準＋オート射撃（離すとビーム/グレネードが向いた先へ）、A=ダッシュ、R1=全開、X=近接、Y=装填、B=着陸/離陸。入力は既存のタッチ/キーボードの**後に加算的に**流し込む（有効なときだけ書く＝クラッシュも既存操作の上書きもしない）。マッピング計算は依存ゼロの純関数`GamepadMap`（デッドゾーン/スティック→単位ベクトル/トリガー閾値）でユニットテスト。※実機コントローラでの挙動はこの環境では未検証（APIコンパイルとCIビルドは通過）。武器切替は当面画面ボタンのまま
- **写真モード**（v2.187）：宇宙のポーズに「写真モード」を追加。選ぶと世界は静止したまま、HUDと演出（天候・バナー・ボスバー・ナビ・カメラ揺れ）が退いて画面が澄む。ドラッグでパン、隅の「遠／近」チップでズーム、「×」で戻る——構図を決めたら端末のスクショで撮る（撮影・保存はOS任せ、アプリはプラットフォームコードを持たない）。カメラ計算は純関数`PhotoCam`（ズーム/パンのクランプ・チップ配置）でユニットテスト。`Overlay.PHOTO`に相乗りするのでシム凍結・発砲抑止は無料。地表ポーズは既に記憶欄で混むため空のポーズ限定（ボタン数は据え置き＝小画面のはみ出しリスクなし）
- **NG+第2の解禁**（v2.190）：2周目クリア以降（clears≥2）、次のランは持物に**周回者専用の装飾品「周回者の徽章」**も携えて出撃——二度目の帰還の証（HP毎秒1.0回復・スタミナ回復+10%・被ダメ-5%の静かな全部載せ）。v2.186の還りのレールと同じ`ngPlusOnly`機構で全プール（ドロップ・壁キャッシュ・市・行商船）から除外＝**未クリア/1周のみの空はバイト単位で不変**。`starterBackpack`に`ngPlus2`段を追加しただけで、レールと徽章は排他ではなく加算（2周でレール＋徽章）。新ItemKind不要（ACCESSORYは既存）、セーブ互換（idは契約）
- **NG+質的アンロック**（v2.186）：周回（同期完了）は数値だけでなく**新装備**を解く。初回クリア以降、次からのランは持物に**周回者専用武器「還りのレール」**を携えて出撃——レールガンの貫通に「拾い手」の磁力を合わせた一挺（＝周回の証）。`ngPlusOnly` フラグでドロップ・壁キャッシュ・惑星の市・行商船のすべてのプールから除外されるので、**未クリア（clears=0）の空はバイト単位で不変**。既存のngClears配線に相乗り、新WeaponDefゼロ・セーブ互換（idは契約、carryで往復）
- **もう一つの結末**（v2.189）：第3の結末（贖いの「還る朝」）には**対になる暗い扉**がある——「刻まれた名を受け入れる」。訪れた星で子を手にかけ（子殺しあり）、複数の星を敵に回した者にだけ、最後の対話で第三の扉がこの姿で現れる。清き道と裁きの道は**排他条件**（一方は子への加害ゼロを、もう一方は流血を要求する）ので、二つが同時に開くことはない。専用エピローグ（忘却APIを持たぬ網が最後に綴じるのは、あなたの名）と実績「星喰い」。既存二択は不変、enum/実績/セーブはすべて末尾追加＝互換維持
- **第3の結末**（v2.185）：同期90%で管制核に立つと、これまで「眠る／漂流する」の二択だった最後の対話に、**隠れた第三の扉**が現れることがある——「網を解いて、朝を返す」。訪れたどの星でも子らを一人も傷つけず（子・雛への加害ゼロ）、複数の星から信頼を得た者だけに開く。眠り続ける網ではなく、疑似人格たちが初めて挨拶を返される朝へ。専用のエピローグと実績「還る朝」。既存二択は不変、行いに依る到達可能な分岐で、enum/実績/セーブはすべて末尾追加＝互換維持
- **濃い外縁III 沈黙の玉座**（v2.184）：九マスの最も深い隅（2-2、二匹目の暴君が守る遠隅）に、一度きりの深部報酬POI「沈黙の玉座」を置いた。二匹の暴君鮫を抜けて玉座に触れると星屑200が積まれる——到達そのものが受領。ラン中一度きりで、エリア再構築でも復活しない（ラッチはセーブに永続、新星系では新品に戻る）。彗星・救助と同じ機構に相乗り＝新種ゼロ・rngストリーム不変、玉座位置はスライスローカルpx（本体との座標系一致を厳守）
- **濃い外縁II**（v2.183）：外縁をもっと厚く。①星系個性を3つ追加——痩せた星系（敵まばら・星屑の実り厚め）／残響の星系（偶数波で異変が起きやすい）／軽い星系（惑星の引力が弱い）。既存の抽選ストリームは不変のまま、独立した第二ストリームで稀に上書きするので同シードの基底draw列は不動。②四隅の主を個性化——どの隅も暴君鮫を戴くのは同じだが、隅ごとに固有の相棒（古の腔棘魚・宙鮫の群れ・雷梶木の槍列・遠隅は二匹目の暴君）を並べ、空の四端が一目で見分く。③危険度勾配の深化——継ぎ目を越えるごとに牙が一枚増え、最も深い隅には二匹目の巨獣が回遊する。すべてrngストリーム不変・新種ゼロ（既存の魚を配置し直すだけ）＝165ピンも海の決定性も不変
- **図録**（v2.182）：討伐図鑑に、象徴的な18種の落ち着いた一文が付いた。図鑑ページの「図録を読む」から、倒して出会った顔だけを読める——森の主、霜の大蟲、獣王、暴君鮫、島鯨、歌鯨、漂流者、三体の王冠ボス……いくつかは既存の読み物と静かに響き合う。出会っていない種・記述のない雑魚は載らない。EnemyDefにloreを足しただけのデータ駆動（描画は純関数RecordsPanel.loreLines＝ユニットテスト可）、英訳はidキーのSpeciesLoreEnに全文（名前の自動英訳とは別系統）
- **静かな拾い手**（v2.181）：①静寂波——11波周期の9波目（9/20/31…）は敵が湧かない。「静寂 — 網が息をつく」の一行だけを残して、補給と装備替えの間合いが波のリズムに刻まれる（残敵を倒し切るまで次は開かない）。②拾い手のSMG——構えているだけでドロップ品が寄ってくる磁石じかけの銃。特性付き武器の第一号だが、特性の配管は v2.35 から全装備欄を見ていたので、新コードはカタログの一行だけ
- **今日の宙域**（v2.180）：検証ランに日替わりの弟が生まれた。タイトル右上「記録」の下の「今日の空」チップから、UTC真夜中ごとに変わる固定シードの空へ（週の空とは別の塩で混ぜるので鏡写しにならない）。日替わりの台帳は週の台帳と別棚（互いを踏まない）。検証ラン/今日の空のゲームオーバー時、結果コード（例: W2947-w9-k77-a3f、チェックサム付き）が自動でクリップボードに載る——チャットに貼れば同じ空を飛んだ誰とでも比べられる。持物のMAPタブに九マスの足あと——この空でどの宙域を踏んだか、いまどこにいるかの小さな図（セーブにも記録）
- **細かい直し**（v2.179）：①SMGの死に武器解消——同弾種のマシンガンに厳密劣位だったのを「軽い懐刀」に（威力12・36発・装填0.7秒＝火力を機動と稼働率で置き換える個性）。②デスクトップの案内の嘘を修正——「マウスで照準」は実装に存在しない。矢印キー+Kが真実。③クラッシュ黒箱にビルド版を刻印（どのdriftが落ちたか分かる）。④README冒頭の2世代前の数値（武器7種・敵16種）を現実（9種・165種）に。store-listingも再刻印。⑤継ぎ目の入場計算をAreaGridの純関数に統合（本体とテストの二重実装を解消）。⑥契約テスト2本——全ストアが引き継ぎに載ること、シムが時計/設定/入力装置を読まないことを、ソースを走査して恒久固定
- **読みかたの頁**（v2.178）：無文字HUDに「読みかた」を。①操作説明に凡例を追加——くさび・リング・細バー・角・星の意味をメニュー内でだけ教える（プレイ画面の文字は増やさない）。②アイコンの意味衝突を解消——残敵は小リング、星屑は四芒星に（ひし形の三重使用が解けた）。③縁マーカーに内側の形——核=角・商=丸・船=横棒・発=上向き・遺=下向き（色と形の二重符号化＝色覚に依存しない）。④惑星コンパスを操作ヒントから分離——方角と距離は道具であって教習ではない
- **濃い外縁**（v2.177）：中央は教え、外縁は牙を剥く。①ボスのローテーションが3種→6種——中ボスの雄（錆の巨人・書庫長・潮の番人）を王冠付きで昇格（錆の王・大書庫長・大潮の番人、いずれも既存の固有技持ち）。②九分割に危険度勾配——中央以外の宙域は狩人と巨獣が増える。③四隅の主——空の四隅にはそれぞれ確定の暴君鮫がお供を連れて待つ（二つの継ぎ目を越えた者だけの目的地）。④新しい星系の個性「多生の星系」——海が一段濃くなり牙も増える（rngストリーム不変＝同シードの海は部分集合則のまま）
- **GCの静音化**（v2.176）：Androidのカクつきの残り火＝ヒープのゴミを毎秒数万個単位で消す。①SpatialGridをHashMap<Long,…>からプリミティブ配列のopen addressingへ——毎tick約5000挿入+数千検索のLongボクシングを殲滅（セル走査順・バケット内順は不変＝シムはバイト同一）。②FlowFieldのBFSキューをArrayDeque<Int>から再利用IntArrayへ——毎秒3回×最大1.5万個のIntegerボクシングを除去。③空域の常時ヒント（惑星/門コンパス）とナビPOIリストの毎フレーム再生成をキー付きキャッシュに（距離は8px刻みで量子化——見た目は不変、GCは沈黙）
- **描画の倹約IV**（v2.175）：シムが毎tick組み直している空間グリッドを描画側にも供給。drawMobs・攻撃予兆リング・ボスバー走査の3箇所が約5000体のモブ全走査をやめ、カメラ/聴取半径の近傍クエリに（間引かれるのは候補集合だけ——カリング判定は不変、tick中に死んだ個体はgetOrNullが黙って飛ばす）。全武器射程内はグリッド完全維持（v2.169のGRID_KEEP=1650px）がカメラ半径を必ず覆うため、画面内の取りこぼしは起きない。性能表示の scene/mob 行で実機の効果を確認できる
- **命綱**（v2.174）：①ラン中断セーブ——Androidがバックグラウンドのアプリを殺すと、最後に手動セーブした地点までランが巻き戻っていた（未セーブなら消滅）。GameScreen.pause()で本物のランだけを静かにチェックポイント（訓練は痕跡を残さない設計、検証ランは週シードから再構築されるため対象外）。②強化カードの抽選をラン毎の記録済みシードに——検証ランは週シード固定で全員同じ提示、将来のリプレイへの布石。③世界再構築3箇所（訓練の出入り・検証ラン・やり直し）のdispose漏れ修正——旧世界の大配列が二重に居座るヒープスパイクを解消。④性能計の区間分割——draw内訳（scene/mob/hud）を2行目に表示、次の削減先を実機数値で特定できる
- **黙る星々**（v2.173）：語り系（世界観ヒント＝難破船の遭難ログ・記憶核の語り・生物のセリフ吹き出し）を既定OFFに。星々は頼まれたときだけ語る——設定でONにすれば従来どおり。すでに自分で切り替えた人の選択はそのまま

> 宇宙は**摩擦ゼロの慣性移動**です。通常移動には速度上限があり、ダッシュだけがその上限を超えて加速できます（流しダッシュ＝巡航の2倍／ボタンダッシュ＝3倍が上限。ただし既に出た速度は上書きされず慣性は保たれる）。手を離しても減速しないため、向きを変える・止まるには逆方向への噴射（逆スティック／逆ダッシュ）が必要です。惑星の地表に降りると摩擦が戻り、足が止まります。

> **惑星への着陸 / 発進**：宇宙で惑星に近づく（約12タイル以内・離れても約3秒の猶予あり）とスキャンカードが出て着陸できます。**キーボードは L、スマホは「スキャンカードのタップ」または緑の「着陸」ボタン**で地表ステージへ。地表では**脱出パッドの上に立って L／「発進」ボタン**で、出発した惑星のすぐ近くの宇宙へ戻ります（往復はランの状態＝HP・弾・強化・装備を引き継ぎます）。
