# drift (Android / Kotlin + libGDX)

**drift** — 慣性で漂う宇宙と、あなたを覚えている星々。

> **世界観（v2.48〜）**: 人類は知恵をAIに預けすぎ、やがてAIにしか保守できない文明になった。
> AIは人類を保全するため惑星規模のサーバー群を築いたが、大同期崩壊で星々は分断され、人類は消えた。
> それでも保全装置は止まらない——保存されたログから、子を、王を、獣を、市を、祈りを出力し続けている。
> あなたが降り立つ星は未開の惑星ではない。**稼働中の人類保全装置**だ。
> 星屑はプロセスが遺す記憶片、市はローカルAPI、危険度はアクセスリスク評価、そして星はログを消さない。
> 真相はゲーム内の読み物（承認者の日記 / 説明負債について / 障害報告書 SYNC-0001 / 保守員の手記 …）に断片として眠っている。
見下ろし型・ウェーブ制ローグライトのサバイバルシューターを、**Android専用ネイティブゲーム**として
Kotlin + libGDX で再実装するプロジェクト。（旧称: ARPG サバイバル）

> 旧 Web 版（Java/Servlet + Canvas/JS + Capacitor）は git タグ `legacy-web-v1.1.1` に保存されています。
> 現行ブランチには旧コードは含まれません（Phase 0 でクリーンに再スキャフォールド済み）。
> 設計の経緯は [docs/superpowers/specs/2026-06-28-android-native-redesign-design.md](docs/superpowers/specs/2026-06-28-android-native-redesign-design.md) を参照。

## モジュール構成

- `core/` — ゲームロジック（プラットフォーム非依存・Kotlin + libGDX + Fleks ECS）
- `desktop/` — LWJGL3 ランチャー（**開発専用**：PCで高速反復）
- `android/` — 配布対象の Android アプリ（APK）

## ゲーム内容

- 7 種の武器（ピストル / ショットガン / マシンガン / ビーム / グレネード / SMG / ライフル — SMG・ライフルは装備で解禁）と近接・ダッシュ・設置壁
- 通常敵 3 種（ゾンビ / スピッター / ストーカー）＋ 中ボス 2 種・ボス 1 種（通常 5 種＋ボス 11 種・計 16 種の攻撃ハンドラ）
- エスカレートするウェーブ（5 の倍数で中ボス、10 の倍数でボス）とスコア
- ウェーブ間の 3 択強化カード（恒久強化・ラン中持続）
- 撃破/被弾の演出（破片・画面シェイク）とゲームオーバー＋リスタート
- タッチ操作（左：移動スティック / 右：アクションボタン）・効果音・ハプティクス
- ハイスコア永続化（最高ウェーブ / 撃破数）
- 日本語 UI（`gdx-freetype` + DotGothic16 同梱・CJK グリフは描画時に逐次生成）
- **Living Planets**：惑星バイオームごとに2つの住人が共存 ——「知性生物は社会を作る。野生動物は生態系を作る。」
  - 知性生物の社会：喋る・守護・突撃・降伏（自然惑星なら獣王の野営地など）
  - 野生動物の生態系：喋らない草食・群れ・捕食者・巣守り・頂点捕食者が、プレイヤーを一律に襲わず逃げる/群れる/狩る/威嚇する
  - 惑星スキャンカード：宇宙で惑星に近づくと、着陸前にその星の気質・聖・兆候と「星があなたを覚えているか」を表示
  - 地表イベントフィード：子を傷つけた・捕食者を退けた等、星の記憶に刻まれた瞬間を短文で通知
  - 惑星記憶サマリ：地表のポーズから「この星の記憶」——事実の一覧と敵意・感謝・乱れの3ゲージを振り返れる
  - 地表ゴール：星ごとに決まる目標チップ（子らを守れ・神獣に触れるな・遺物など）を目的行の下に表示
  - 再訪の配置変化：敵対の星はパッド脇に見張りを置き、感謝の星は群れが逃げず実りが増え、乱れた星は獲物が減り捕食者が飢える
  - 宇宙の永続記憶：アプリを閉じてもゲームオーバーしても星はあなたを覚えている（ポーズの「宇宙の記憶を消す」で全忘却）
  - 離陸の餞別：主討伐・遺物・民の感謝・子らの無事に応じて星が資材・弾薬・回復を持たせる——子を殺した星は何もくれない
  - 記憶の見た目：敵対の星は宇宙から赤みを帯びて見え、感謝の星は緑に灯る。再訪の挨拶も星ごとに声色が変わる
  - 着陸/離陸の暗転フェードと専用SE（着陸・発進・惑星スキャン）。通常移動は静止からゆっくり加速する重みのある操作感
- **インベントリ＆装備**（v2.33）：I キー / 持物ボタンで開くと世界が 0.01 倍速に。装備 7 欄（推進器・防具・遠距離・近距離・装飾×3）をタップで持物と交換、遠距離欄の交換＝武器切替。持物一覧・踏破マップ（通った場所だけ描画）・その場セーブ（やられると消える）。敵は稀に装備アイテムを落とし（ボスは確定）、**OC スラスター**装備時は全開ボタン / O で加速3倍・巡航2倍・スタミナ消費2倍のフルスロットル
- **消費アイテム＆読み物**（v2.34）：カタログは約 50 点に拡充。回復・スタミナ・スティム・煙幕・資材・弾薬などの**消費アイテム**はアイテムタブのタップで使用（【使】印）。**読み物**（【読】印）は獣王の壁画・氷の民の子守唄・死せる星の碑文など、六つのバイオームの物語を運ぶ——ドロップは消費 55% / 装備 30% / 読み物 15% の重み付き。着陸は**スキャンカードのタップでも可能**になり、着陸可能域が2倍・猶予約3秒に拡大（Android の着陸しやすさ改善）
- **UI第2弾：ハブ&アーク**（v2.55）：タッチ操作を**ダッシュ中心のハブ&アーク配置**へ——右下に大きなダッシュ（半径1.5倍のハブ）、その外周弧に**近接・武器・装填**、第2弧に**壁・全開**が並び、親指の一払いで全アクションに届く。**持物は右上**（⏸ボタンの下）へ退避して戦闘系と分離。エイムガイドも弧と干渉しない位置へ。上部HUDは**1行ステータス帯+2カラム**に再設計——「同期汚染 N　残プロセス M　宙域安定 S%」が細い帯の中に収まり（旧バッジのように下へ垂れてHPバーと重ならない）、HPバー/スタミナは左カラム、武器パネルは右カラム（⏸の手前で必ず止まる）、統計行は全幅+自動縮小。武器名も残り幅に自動フィット
- **モダンUI**（v2.54）：スマホ実機での「文字はみ出し・重なり・ボタン見切れ」を一掃。**ヒントは全てガラス調パネル**（半透明ダーク+淡い縁取り）でHUD帯の下に表示され、長文は**自動縮小**して画面幅に必ず収まる（トースト・待機ガイド・ゲートコンパス・地表目標・チップ・オンボーディングすべて同経路）。HUD上部帯には可読性スクリム。**タッチボタンは画面端から内側へ計算するグリッド配置**に刷新——+21%大型化・一定間隔・セーフマージンで**どのアスペクト比でも見切れゼロ**（回帰テストで保証）。見た目もダークガラスの円盤+機能色リング（ダッシュ=シアン/近接=レッド/着陸=グリーン…）に統一し、円からはみ出すラベルは自動縮小
- **旧式戦闘訓練**（v2.53・長期移行）：ウェーブ制を**別モードとして切り出した**——ポーズメニューの「旧式戦闘訓練」で、固定シードの模擬戦アリーナに入る。装備・資材はそのまま持ち込めるが、**成果は持ち出せない**（入場時の状態が退場時に完全復元）。訓練中は着陸・跳躍・セーブ・遭難ログすべて無効、HUDバッジは懐かしの**「ウェーブ(旧式)」**表示に戻り、死んでも記録更新なし・セーブ消滅なし・遺品も残らない——シミュレーションが残すのは腕前だけ。通常モードの進行は星間同期復旧度・宙域安定・呼び名が担う
- **同期乱流と復旧度**（v2.52・中期移行）：スポナーは **DesyncSurgeSystem** に改名——「ウェーブ」は壊れた保全ネットワークが未処理プロセスを周期的に放出する**同期乱流**として実装レベルでも再定義された。**汚染が深まるほど深層のプロセスが漏れ出す**：序盤の乱流は基本3種だけ、乱流が進むごとに敵名簿の「地層」（v2.39層→v2.41層→保守機構層）が1種ずつ順に解禁される。進行の真の軸として**星間同期復旧度**を導入——跳躍した星系数・訪れた星・信頼を得た星から算出（上限99%——最後の1%は、仕様書を読めた人々と共に失われた）。**復旧度60%以上でジャンプゲートが保守員の署名を認識し、鍵の断片が2個で開く**。航海日誌に「星間同期復旧 X%」を表示
- **言葉のほころびと遭難ログ**（v2.51）：住民の台詞に**疑似人格の「ほころび」**が混ざるようになった——王は「去れ。……本プロセスは、そう告げる役目だ」と口を滑らせ、護衛は「子を守れ。第4条だ」とポリシーを漏らし、命乞いは「終了させないでくれ。まだ出力の途中だ」になる。子殺しへの糾弾には「星は消去APIを持たない」、再訪の歓迎には「あなたの署名は白名簿に載っている」。そして**難破船に近づくと遭難ログを一度だけ受信**する——「救難信号 7,040日目。今日から数えるのをやめる」「最後の通信: 『灯りは点けたままにしておく』。灯りは、点いている」（難破船ごとに決定的・8種）
- **同期汚染**（v2.50）：ウェーブ制を世界観に接続する再解釈第1弾——敵の波は「襲撃」ではなく、**壊れた保全ネットワークが未処理の人格断片や防衛プロセスを周期的に出力している現象**になった。HUDバッジは「ウェーブ N」→**「同期汚染 N」**、残り敵数は「残プロセス」、その下に**宙域安定 %**（汚染が深まるほど低下、5%で下げ止まる——機械は持ちこたえる）。強化カードは「クリア報酬」から**「同期汚染を収束 — 保守パッチを選択」**へ。ゲームオーバーとベスト記録は「汚染深度」。航海日誌には**呼び名**が加わる：星々の記憶の集計から 異邦人→巡回者／星還し／遺物持ち／王殺し／獣狩り、そして子を殺した者は何をしても**星喰い**（最悪の行いは上書きされない）。地表は従来どおり無限湧きなし（住民は社会、スポナーは宇宙のみ）
- **段階開示**（v2.49）：ジャンプゲートで星系を進むほど、**記憶核の語りがプレイヤー自身に迫っていく**——第1星系では定常ログ、第2〜3星系では「訪問者の署名を照合中。過去の保守員名簿に部分一致がある」、第4星系以降は「照合完了。あなたの人格ログの作成日は、あなたの記憶する誕生日より後だ」「おかえりなさい、最終保守員」。航海日誌の肩書も 漂流者（未照合）→（署名照合中）→**最終保守員（照合済み）** と変わる。読み物に**保守員の手記 III**（真相を知った後の最後の頁）を追加
- **惑星サーバー**（v2.48）：世界観の真相を実装。全地表に**記憶核**（暗い石碑にゆっくり呼吸する灯り）が立ち、近づくと星ごとに決まったサーバーログを一度だけ語る——「この森はバックアップである」「『おかえり』を再生する相手を3千年待っている」。宇宙には**保守機構**が加わる：保守ドローン・索引虫・検疫体、そして中ボス**監査体**（知能0.9・ドローン召喚・回避行動）。ウェーブ 7/16/34…は**清掃プロトコル**——保守機構だけが物量1.2倍で展開する新イベント。読み物12点（承認者の日記・説明負債・最後の委員会議事録・障害報告書 SYNC-0001・人類保全ポリシー・星屑の正体・保守員の手記 I/II など）と装備4点（トルクブレード・診断レンズ・保守機構の外殻・修復パッチβ）を追加
- **武器合成と道案内**（v2.47）：**同じ武器を2つ持っていると合成できる**——アイテムタブで【合成可】の付いた行をタップすると2つが1つの**「+1」**に鍛え直される（威力+12%・連射/装填もわずかに向上、**最大+3**まで重ね掛け可）。強化品はセーブ・持ち越しにも対応。そして**初回起動時だけ**、移動→照準→ダッシュ→着陸を32秒かけて順に案内する**道しるべ**が画面下に出る（2回目以降は二度と出ない）
- **航海日誌・遺品回収・難破船**（v2.46）：インベントリに**「記録」タブ**（航海日誌）——現在の星系・ウェーブ・撃破・星屑・ゲート鍵、自己ベスト、そして**訪れた星々があなたをどう覚えているか**を一覧。やられると**遺品**が残る：所持星屑の半分とゲート鍵の全てが**倒れた場所に漂い続け**、次のランで回収できる（地表で倒れた場合は宇宙の出発点近くに漂着）。宇宙には**難破船が2〜3隻**——折れた船体に非常灯が明滅し、武器ケース・星屑の束・救急パックを**漂流者の哨戒**が守る
- **賞金首・イベントウェーブ・星の依頼**（v2.45）：ウェーブに**リズムが生まれた**——4/11/18…は**賞金首**（『錆のカラス』『宵のスズ』など二つ名持ちの精鋭が出現、討伐で星屑の山が弾ける・報酬はウェーブと共に増加）、3/8/13…は**大群**（物量1.6倍・出現間隔短縮）、5/17/29…は**磁気嵐**（敵はほぼ全員ダッシュで荒ぶるが、星屑ドロップ2倍）。各ウェーブ開始時に告知トーストが流れ、**ボス/中ボス到来も「強大な気配」として予告**（ボスの格）。地表では**星の依頼**：惑星ごとに決まった討伐依頼（精鋭1〜3体 or 外敵8〜16体）がチップに進捗表示され、達成して離陸すると星屑で報酬——ただし子を殺した相手には払われない
- **ジャンプゲート**（v2.44）：宇宙のどこかに**次の星系へ跳ぶゲート**が必ず1基ある（シードから決定的に配置・水色のリング）。**中ボス級以上を倒すと「ゲート鍵の断片」**（水晶のかけら）が1つこぼれ、**3つ集めてゲートに接近すると [L] / ゲートタップで跳躍**——星系番号が進み、惑星も敵もゲートも新しく生成され、装備・資材・ウェーブ数はそのまま持ち越し（実質 New Game+）。跳躍で船体は全回復。HUDには鍵の所持数（集め終わるとゲートへの方角と距離）を常時表示。断片はセーブにも保存
- **星屑と惑星の市**（v2.43）：撃破で必ず**星屑**（通貨・金の四芒星）がこぼれ、HUDに所持数を表示。**友好的な星の地表ではインベントリに「市」タブ**が開き、星ごとに決まった6枠の品を星屑で購入できる——**感謝が深い星ほど安く（最大40%引き）、敵意が0.5を超えた星は取引そのものを拒否**。評判に実利が生まれた。スキャンカードには**危険度（★1〜4）**を表示：バイオームの脅威+その星の恨みで加算。星屑はセーブ・惑星往復でも保持
- **縮地と階級**（v2.42）：**照準武器（ビーム/グレネード）の不発バグを修正** — 離した瞬間の入力を約0.3秒バッファし、シムが進んでいないフレームやクールダウン明けでも確実に発射。**重力下（惑星地表）のダッシュは残像を残す縮地**に（瞬間移動・スタミナ消費・短いクールダウン）、宇宙では従来のスラスター。近接武器に**衝角メイス**（吹き飛ばし2倍）・**吸収ブレード**（与ダメ15%回収）・**収穫鎌**（全周刈り+吸収）・**単分子エッジ**（鋭利・ノックバック半減）。敵は**階級が見た目と実力に出る**——王の金冠・長老の鉢巻・守護者の盾・エリートの肩甲・成り上がりのレベルピップ、そして**中ボス/ボスは地位相応のレベルで出現し全技を使い、戦闘知能も高い**（従来はボスすらレベル1で最初の1技しか使っていなかった）
- **拡がる敵と武器庫**（v2.41）：攻撃ハンドラに**スパイラル**（二重リングの弾花）・**スプレー**（乱射弾幕）・**ツインショット**（平行二連）・**ショックウェーブ**（無敵時間中でも吹き飛ぶ衝撃波）の4種を追加し、それを使う敵5種——結晶生物**プリズム**、**スプレイヤー**、回避の達人**デュエリスト**、扉を守る**ウォーデン**、攻城中ボス**アーティラリー**——が参戦。アイテムも12点追加：高密度メディゲル・戦闘刺激剤・大型弾薬ケースなどの消費6点、装甲下地・軽合金フレーム（移動+12%だが被ダメ+5%のトレードオフ）・集束レンズの装飾3点、破砕ハンマー、読み物2点（漂流者の通信記録・結晶の観測ノート）
- **拾い集める武器庫**（v2.40）：マグマの熱を**約20秒で1ダメージ**の“じわじわ効く圧力”に緩和（隣接ブロック数で増えることもなし）。**チャージビームは太いほど着弾爆発も大きく**（半径・威力とも最大2倍）。銃に**弾道バリアント9挺**を追加——速射/大口径/**誘導**（敵を追尾）ピストル、弾幕ストーム/精密マシンガン、ワイド/スラッグショットガン、誘導SMG、弾速2倍のレールライフル。銃は計26挺になり、敵ドロップに加えて**壁の塊を掘り抜くと稀に武器キャッシュ**（2%）が見つかる
- **画面全域マッピングと生きた宇宙**（v2.39）：踏破マップは**画面に映った範囲がそのまま記録**される方式に。惑星は**1宙域6〜8個**・半径56〜200の多彩なサイズになり、未接近時は**最寄りの惑星の方角と距離**を常時表示（着陸迷子の解消）。壁素材に**結晶とヴォルカニックアッシュ**、宇宙の敵に**ランサー・モーター蟲・盾持ち・マイト・マークスマン・サッパー**の6種（攻撃・体格・性格・知能すべて別物）。**ビームは照準中にチャージ**され、長く構えるほど太く強く（最大2倍・貫通は健在）。近接武器に**攻撃範囲拡大（大鉈・ハルバード）と飛ぶ斬撃（振動ブレード）**。インベントリの装備タブから**近接=ボタン/銃=右スティックの役割を入替**可能
- **新武器種と手に取れるドロップ**（v2.38）：**SMG**（超速連射）と**ライフル**（単発高威力）が装備経由で解禁（グレード込みで計17挺）、マシンガンは**80発マガジン**に。装備に**対衝撃フレーム / ジャイロスタビライザー**（激突ダメージ無効）、巡航/バーニアスラスター、偵察スーツなどを追加。ドロップ品は**種類ごとの形**に——弾はカートリッジ、回復は十字ケース、装備は銃・刃・盾のシルエット、読み物は本。宇宙では**惑星そのものをタップで着陸**でき、未接近時も画面上部に着陸の手順を常時表示
- **武器グレードと装備の見た目**（v2.37）：銃はグレード制に——同じ「種類」のまま**威力・連射・装填速度・ブロック破壊力・爆風の広さ**が変わる上位機（ピストルM2/M3・マシンガン改・重機関銃・解体ショットガン・高出力ビーム・拡散グレネードなど計13挺）。**ビームとマシンガンは弾薬無限**に。装備を替えると**キャラの見た目も変化**——武器種ごとに銃のシルエットが変わり、防具でスーツの色味が変わり、OCスラスターの噴射は青く燃える
- **漂う住人と激突の代償**（v2.36）：宇宙は最初から棲まれている——**30体の漂流者**が慣性のままに漂い、近づけば牙を剥く（遠くの漂流者がウェーブ進行を止めることはない）。また**高速で壁や惑星に激突すると速度に応じて1〜5ダメージ**（軽い接触は無傷のまま）
- **特殊効果の装備と時限消費**（v2.35）：**耐熱スーツ / 遮熱フォイル**はマグマダメージを完全カット、**防寒スーツ / グリップソール**は雪の減速と氷の滑りを無効、**回収磁石**はドロップ品を3倍の距離から回収、**自己修復パッチ**はHPを毎秒回復。消費側には**耐熱/防寒コーティング（60秒）・携行磁石（45秒）・応急再生パック（20秒）**の時限版。あわせて装備名を落ち着いたトーンに整理（獣王の大牙→大型獣の牙 など）

## 操作

| 操作 | キーボード（開発） | タッチ（Android） |
|------|----|----|
| 移動 | WASD / 矢印 | 左スティック |
| 射撃 | K（押しっぱなし） | FIRE |
| 近接 | J | ML |
| ダッシュ（向き方向・強加速・スタミナ大） | Shift | DASH ボタン |
| 流しダッシュ（スティック方向・微加速・スタミナ極小） | — | 左スティックを大きく倒す |
| リロード | R | RL |
| 壁設置 | F | WALL |
| 着陸 / 発進（惑星 ⇄ 宇宙） | L | **惑星かスキャンカードをタップ**／緑の着陸ボタン |
| 武器切替 | 1〜5 | WPN（順送り） |
| インベントリ（装備/持物/マップ/セーブ） | I | 持物ボタン |
| フルスロットル（OC スラスター装備時） | O（押しっぱなし） | 全開ボタン |
| 強化カード選択 | 1 / 2 / 3 | （数字キー） |
| リスタート | R（ゲームオーバー時） | — |

> 宇宙は**摩擦ゼロの慣性移動**です。通常移動には速度上限があり、ダッシュだけがその上限を超えて加速できます（流しダッシュ＝巡航の2倍／ボタンダッシュ＝3倍が上限。ただし既に出た速度は上書きされず慣性は保たれる）。手を離しても減速しないため、向きを変える・止まるには逆方向への噴射（逆スティック／逆ダッシュ）が必要です。惑星の地表に降りると摩擦が戻り、足が止まります。

> **惑星への着陸 / 発進**：宇宙で惑星に近づく（約12タイル以内・離れても約3秒の猶予あり）とスキャンカードが出て着陸できます。**キーボードは L、スマホは「スキャンカードのタップ」または緑の「着陸」ボタン**で地表ステージへ。地表では**脱出パッドの上に立って L／「発進」ボタン**で、出発した惑星のすぐ近くの宇宙へ戻ります（往復はランの状態＝HP・弾・強化・装備を引き継ぎます）。
