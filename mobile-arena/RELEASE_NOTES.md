# Wave Arena v1.0.0

見下ろし型サバイバル・アクションシューター（ウェーブ制ローグライト）。
サーバ不要の静的PWA。Capacitor で Android/iOS ネイティブ化も可能。

## 遊び方

- ローカル: `cd mobile-arena && npm run serve` → `http://localhost:8080/`
- 公開版: GitHub Pages（リポジトリ Settings → Pages → Source =「GitHub Actions」）

### 操作
- **キーボード**: 移動 `WASD`/矢印、射撃/投擲 `K`、近接 `J`、リロード `R`、ダッシュ `Shift`、壁 `F`、武器 `1`〜`5`、セーブ `P`、ロード `L`、ポーズ `Esc`
- **タッチ**: 左スティック=移動 / 右スティック=照準＋射撃。ボタン（近接・ダッシュ・リロード・武器〔長押しでラジアル〕・壁・ポーズ・設定）。横向き推奨。

## 主な機能

- **タイトル画面**: はじめから / つづきから（セーブロード＋到達ステージ選択）/ 設定 / スコア / モード選択
- **モード**: ステージ（全4面）／エンドレス（全ステージ制覇で解放）
- **ステージ進行**: 5ウェーブごとに専用マップへ切替。難易度は敵 AI の「反応速度・回避・判断・技のレパートリー」を段階強化（HP/ダメージの水増しはしない）
- **ウェーブ制ローグライト**: 殲滅→強化カード選択。中ボス／ボス、キルカム演出
- **タッチ完結UI**: ポーズ／セーブ・ロード（3スロット）／設定の overlay、武器ラジアル、セーフエリア対応、タッチUI常時表示制御、A11y（aria / 44px タップ）
- **永続化**: localStorage（Native は Preferences ミラー）。セーブ schema v4（mode/stage）＋後方互換 migrate
- **ネイティブ統合（任意・Web は no-op）**: ハプティクス、Android 戻るボタン、中断/復帰の自動ポーズ、ステータスバー/スプラッシュ
- **PWA**: オフライン起動（Service Worker `wave-arena-v16`）、ホーム画面追加対応

## 品質

- 純関数中心のユニットテスト **138件**（`node --test`）、全モジュール構文チェック、CI（`mobile-arena-ci`）で自動実行。

## 既知の制約

- APK/IPA ビルドには各 OS SDK が必要（生成物はリポジトリに含めない）。
- StatusBar の色制御は Android 15/16+ の edge-to-edge で限定的（破綻回避を目標）。
- オンライン同期・認証は範囲外（ローカル完結）。
